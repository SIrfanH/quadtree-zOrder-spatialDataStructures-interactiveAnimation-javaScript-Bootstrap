MX Quadtree
How To Use

1. Used for Storing Lines and Polygon edges. Vertex Based version of the PM Quadtree Family.

2. In every insert decomposition happens until the order level is reached.

3. The lines are added to every quandrants that they pass.

1. Insert lines by drawing with your mouse on the canvas or select saved lines.

2. Select Window Seach then draw a rectangle and click animate.

3. Select KNN Search then select K from the slider then click anywhere on canvas to find K-Nearest Neighbor.



PM1 Quadtree
How To Use

1. Used for Storing Lines and Polygon edges. Vertex Based version of the PM Quadtree Family.

2. Leaf nodes can contain at most one vertex.

3. If a leaf node contains a vertex then it can only contain q-edges that include that vertex.

4. At most one edge can pass from a block that does not contain a vertex.

1. Insert lines by drawing with your mouse on the canvas or select saved lines.

2. Select Window Seach then draw a rectangle and click animate.

3. Select KNN Search then select K from the slider then click anywhere on canvas to find K-Nearest Neighbor.



PM2 Quadtree
How To Use

1. Used for Storing Lines and Polygon edges. Vertex Based version of the PM Quadtree Family.

2. Leaf nodes can contain at most one vertex.

3. If a leaf node contains a vertex then it can only contain q-edges that include that vertex.

4. If a leaf node does not contain a vertex then only q-edges that have a common vertex outside the block can pass through it.

1. Insert lines by drawing with your mouse on the canvas or select saved lines.

2. Select Window Seach then draw a rectangle and click animate.

3. Select KNN Search then select K from the slider then click anywhere on canvas to find K-Nearest Neighbor.



PMR Quadtree
How To Use

1. Used for Storing Lines and Polygon edges. Edge base version of the PM Quadtree Family.

2. Decomposition happens only once if maximum capacity of a block is exceeded.

3. It is possible for a block to contain more edges than the specified capacity

4. Different insertion sequences create different tree shape.

1. Insert lines by drawing with your mouse on the canvas or select saved lines

2. Select Window Seach then draw a rectangle and click animate

3. Select KNN Search then select K from the slider then click anywhere on canvas to find K-Nearest Neighbor.



Z-Order Curve
How To Use

1. Space Filling Curves if infinitely drawn has the ability to fill the entire space.

2. They visit every point in space exactly once and does not intersect itself.

3. Z Order index (Morton Code) of a point is aquired by interleaving X and Y coordinates of it.

1. Select Curve order using the slider to see the curve filling the space.

2. Draw a rectangle with your mouse on the canvas for Window Search.
