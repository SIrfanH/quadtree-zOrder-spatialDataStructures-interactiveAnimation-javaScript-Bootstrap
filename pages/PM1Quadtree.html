PM1 Quadtree
How To Use

1. Used for Storing Lines and Polygon edges. Vertex Based version of the PM Quadtree Family.

2. Leaf nodes can contain at most one vertex.

3. If a leaf node contains a vertex then it can only contain q-edges that include that vertex.

4. At most one edge can pass from a block that does not contain a vertex.

1. Insert lines by drawing with your mouse on the canvas or select saved lines.

2. Select Window Seach then draw a rectangle and click animate.

3. Select KNN Search then select K from the slider then click anywhere on canvas to find K-Nearest Neighbor.
