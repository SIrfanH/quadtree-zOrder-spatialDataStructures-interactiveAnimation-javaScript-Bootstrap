PMR Quadtree
How To Use

1. Used for Storing Lines and Polygon edges. Edge base version of the PM Quadtree Family.

2. Decomposition happens only once if maximum capacity of a block is exceeded.

3. It is possible for a block to contain more edges than the specified capacity

4. Different insertion sequences create different tree shape.

1. Insert lines by drawing with your mouse on the canvas or select saved lines

2. Select Window Seach then draw a rectangle and click animate

3. Select KNN Search then select K from the slider then click anywhere on canvas to find K-Nearest Neighbor.
